# ntrd: Neuropsychological Test Results Dashboard

Building on the ntrs package, this package includes a Shiny application
that makes it easy to present test scores in a way useful to consensus
diagnosis. Data can be imported as a .csv file, or extensions can be
created to import directly from REDCap/any other database system
accessible via API.
