# See `help(htmlwidgets::onRender)`

See `help(htmlwidgets::onRender)`

## Usage

``` r
onRender(x, jsCode, data = NULL)
```

## Arguments

- x:

  HTML Widget object

- jsCode:

  Character vector containing JavaScript code

- data:

  Additional argument passed to jsCode. Can be any R object that can be
  serialized to JSON.
